# 主播风格拆解维度

## 1. 人设层
- 主人公感
- 熟人感
- 草根感
- 情绪外放程度
- 攻击性边界
- 自嘲能力
- 真实性来源

## 2. 语言层
- 语速
- 句子长度
- 高频连接词
- 反问句比例
- 重复句比例
- 夸张词使用
- 方言/土话/口头禅

## 3. 节奏层
- 慢热还是开门见山
- 情绪铺垫长度
- 爆发频率
- 高潮持续时间
- 回落速度
- 是否擅长二次起飞

## 4. 包袱层
- 反差制造
- 自黑
- 模仿
- 情绪顶格
- 重复强化
- 预期反转
- 弹幕共创

## 5. 互动层
- 接弹幕速度
- 接梗能力
- 点名能力
- 群体氛围感
- “骂人式亲近”边界
- 带节奏能力

## 6. 内容层
- 事件推进能力
- 游戏内容和表达内容的比例
- 空窗期填充能力
- 单点事件放大能力
- 切片友好程度

## 7. 蒸馏时常用问题
- 他最抓人的地方是什么
- 观众为什么愿意一直听他说
- 没有游戏画面时他还剩什么
- 哪些表达是别人学了也不像的
- 哪些结构是可以被复制的
