# xuxubaby-style-distiller

一个专门用于**蒸馏旭旭宝宝直播风格**的 OpenClaw Skill。

它不是粉丝向夸夸文工具，而是把旭旭宝宝的直播吸引力拆成**可分析、可复用、可迁移**的方法组件：
- 人设为什么立得住
- 情绪为什么能带起来
- 观众为什么愿意一直听
- 哪些东西能学，哪些东西不能硬抄

## 能做什么

这个 skill 适合做：
- 旭旭宝宝直播风格分析
- 主播人设拆解
- 语言风格蒸馏
- 情绪节奏拆解
- 搞笑机制总结
- 直播互动方法分析
- 切片文案模板提炼
- 短视频口播模仿框架
- 主播风格迁移到其他赛道
- 带素材的直播片段拆解

## 适用场景

当用户提出类似需求时触发：
- 分析旭旭宝宝这个人
- 拆解旭旭宝宝直播风格
- 学习旭旭宝宝的话术
- 蒸馏旭旭宝宝的人设
- 做旭旭宝宝风格模仿
- 拿旭旭宝宝风格做短视频/直播脚本
- 给我一套旭旭宝宝风格的切片模板
- 帮我分析这段录屏为什么有节目效果

## 默认分析维度

当前默认按 **10 个维度** 输出：

1. 人设定位
2. 语言风格
3. 情绪曲线
4. 搞笑机制
5. 互动方式
6. 节奏控制
7. 内容结构
8. 可复用模板
9. 模仿边界
10. 短视频转译能力

## 建议阅读顺序

如果是第一次看这个仓库，建议按这个顺序：
1. `README.md`：先看 skill 的目标和适用场景
2. `SKILL.md`：看默认输出框架和调用方式
3. `references/source-notes.md`：先看公开资料证据底稿
4. `references/analysis-example.md`：看基于公开资料的完整分析示例
5. `references/video-sample-analysis.md`：看基于 B 站公开视频样本的人物蒸馏示例
6. `references/style-dimensions.md`：补充更细的拆解维度
7. 其他模板文件：按需求拿去复用

## 输出特点

不是泛泛而谈的“接地气、幽默、真实”，而是重点回答：
- 他是怎么做到的
- 为什么观众会持续看
- 哪些地方能复用
- 哪些地方只能借鉴不能硬抄
- 如果迁移到短视频，应该怎么改写

## 核心提醒

学旭旭宝宝，重点不是学嗓门，也不是学几个口头禅。
重点是学：
- 情绪调度
- 熟人感
- 节奏推进
- 包袱设计
- 把小事讲成大戏的能力
