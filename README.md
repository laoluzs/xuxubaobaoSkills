# xuxubaby-style-distiller

一个专门用于**蒸馏旭旭宝宝直播风格**的 OpenClaw Skill。

## 能做什么

这个 skill 适合做：
- 旭旭宝宝直播风格分析
- 主播人设拆解
- 语言风格蒸馏
- 情绪节奏拆解
- 搞笑机制总结
- 直播互动方法分析
- 切片文案模板提炼
- 短视频口播模仿框架

## 适用场景

当用户提出类似需求时触发：
- 分析旭旭宝宝这个人
- 拆解旭旭宝宝直播风格
- 学习旭旭宝宝的话术
- 蒸馏旭旭宝宝的人设
- 做旭旭宝宝风格模仿
- 拿旭旭宝宝风格做短视频/直播脚本

## 默认分析维度

1. 人设定位
2. 语言风格
3. 情绪曲线
4. 搞笑机制
5. 互动方式
6. 节奏控制
7. 内容结构
8. 可复用模板

## 输出特点

不是泛泛而谈的“接地气、幽默、真实”，而是重点回答：
- 他是怎么做到的
- 为什么观众会持续看
- 哪些地方能复用
- 哪些地方只能借鉴不能硬抄

## 适合搭配的产出

- 主播风格分析报告
- 直播复盘文档
- 口播模仿脚本
- 切片标题和文案模板
- 短视频人设参考稿
